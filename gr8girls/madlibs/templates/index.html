Getting Started

Here are the files you need to download and save. To get started, follow these steps:

  1. Open the file for editing by clicking the computer icon in the lower left, and selecting "Home folder". Open public_html, and click on MyMadLibs.py to edit it.
  2. What is open now just lets you edit the program. To run it, click on the "K" menu in the bottom left. Under System, open Konsole.
  3. In the console, type exactly cd public_html to take yourself to the public_html folder.
  4. To run the example Madlib, or to run the Madlib as you change it, type python2.5 MyMadLibs.py
  5. If you want to start over again, you can download and save the MyMadLibs.py file to the public_html folder, overwriting the old file.
